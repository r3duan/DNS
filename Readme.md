# DNS

How DNS works: a complete, step-by-step guide to the Domain Name System — notes taken while learning how domain names get resolved into IP addresses, from the resolver all the way down to the authoritative server, plus the local mechanisms that can bypass that process entirely.

## Contents

- [How DNS Query Resolve](./How%20DNS%20Query%20Resolve.md) — the full journey a DNS query takes: your computer's cache, the resolver, root servers, TLD servers, and the authoritative server that holds the real answer.
- [The Hosts File](./The%20Hosts%20File) — how the local hosts file overrides DNS entirely, where it lives on each OS, and practical uses like local development, connectivity testing, and blocking sites.

## Why this repo exists

Keeping these as versioned markdown notes instead of scattered docs — easy to search, easy to update as I learn more, and easy to share.
